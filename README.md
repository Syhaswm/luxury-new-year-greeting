# 运行和部署你的 AI Studio 应用

本项目包含在本地运行你的应用所需的一切。

在 AI Studio 中查看你的应用：https://ai.studio/apps/drive/1aKnO5jaEkqggtI21k25GtA1SWJrd-1AA

## 本地运行

**前置条件：** Node.js


1. 安装依赖：
   `npm install`
2. 在 [.env.local](.env.local) 中将 `GEMINI_API_KEY` 设置为你的 Gemini API 密钥
3. 运行应用：
   `npm run dev`
